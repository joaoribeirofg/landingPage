To-do List

A To-Do List é uma aplicação interativa criada com HTML, CSS e jQuery, permitindo gerenciar tarefas de forma simples e prática

Ela possui uma interface básica, onde o usuário pode adicionar novas tarefas por meio de um campo de entrada de texto, e essas tarefas são exibidas como itens em uma lista dinâmica.

Cada tarefa adicionada pode ser marcada como concluída ao clicar diretamente sobre ela, o que aplica um efeito de texto riscado, indicando visualmente que a tarefa foi realizada.
